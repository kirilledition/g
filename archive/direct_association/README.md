# Direct Association Archive

This directory stores historical direct PLINK-style linear/logistic association code and related artifacts.

Status:

- Reference-only.
- Not importable from the active `g` package public surface.
- Not part of active CLI commands.
- Not part of the active CI test target.

Contents:

- `src/g/compute/`: archived direct association kernels.
- `tests/`: archived direct-association tests and parity flows.
- `scripts/`: archived direct-association benchmarks and profiling utilities.

To resume this path in the future, copy needed components back into active modules and reintroduce public API/CLI surfaces intentionally.
